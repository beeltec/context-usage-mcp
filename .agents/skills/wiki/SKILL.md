# Project Wiki

Use the [Open Knowledge Format](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md) to manage durable project context.

Use `docs/wiki` as the wiki root. Create the directory and its initial Open Knowledge Format files when they do not exist. Preserve existing knowledge and update the smallest relevant set of pages.
